# uuid {#uuid}

Gera um UUID aleatório (Identificador exclusivo universal).

## Categoria

String

## Sintaxe da função

`uuid()`

## Parâmetros

Esta função não requer parâmetros.

## Assinatura e tipo retornado

`uuid()`

Retorna uma string.

## Exemplo

`uuid()`

Retorna &quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;.s
