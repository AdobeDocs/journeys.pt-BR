# uuid {#uuid}

Gera um UUID aleatório (ID exclusiva universal).

## Categoria

Cadeia de caracteres

## Sintaxe da função

`uuid()`

## Parâmetros

Essa função não exige parâmetros.

## Assinatura e tipo retornado

`uuid()`

Retorna uma string.

## Exemplo

`uuid()`

Retorna &quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;.s
