# contém {#contain}

Verifica se a segunda string de argumento está contida na primeira string de argumento.

## Categoria

Cadeia de caracteres

## Sintaxe da função

`contain(<parameters>)`

## Parâmetros

* string

## Assinatura e tipo retornado

`contain(<string>,<string>)`

Retorna um booleano.

## Exemplo

`contain("rowing is great', "great")`

Retorna true.
