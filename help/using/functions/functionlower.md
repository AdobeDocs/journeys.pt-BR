# lower {#lower}

Retorna uma versão em minúsculas do parâmetro.

## Categoria

String

## Sintaxe da função

`lower(<parameter>)`

## Parâmetro

* sequência de caracteres

## Assinatura e tipo retornado

`lower(<string>)`

Retorna uma string.

## Exemplo

`lower("A")`

Retorna &quot;a&quot;.
