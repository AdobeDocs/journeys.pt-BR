# trim {#trim}

Remove espaços de start e fim.

## Categoria

String

## Sintaxe da função

`trim(<parameters>)`

## Parâmetro

| Parâmetro | Tipo |
|-----------|------------------|
| string | string |

## Assinatura e tipo retornado

`trim(<string>)`

Retorna uma string.

## Exemplo

`trim(" Hello ")`

Retorna &quot;Hello&quot;.
