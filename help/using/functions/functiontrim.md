# trim {#trim}

Remove espaços iniciais e finais.

## Categoria

String

## Sintaxe da função

`trim(<parameters>)`

## Parâmetro

| Parâmetro | Tipo |
|-----------|------------------|
| string | string |

## Assinatura e tipo retornado

`trim(<string>)`

Retorna uma string.

## Exemplo

`trim(" Hello ")`

Retorna &quot;Olá&quot;.
