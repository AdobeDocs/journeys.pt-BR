# isEmpty {#isEmpty}

Retorna true se a string no parâmetro não tiver nenhum caractere.

## Categoria

String

## Sintaxe da função

`isEmpty(<parameters>)`

## Parâmetros

* string

## Assinatura e tipo retornado

`isEmpty(<string>)`

Retorna um booleano.

## Exemplo

`isEmpty("")`

Retorna true.

`isEmpty("Hello World")`

Retorna false.
