# upper {#upper}

Retorna uma versão em maiúsculas do parâmetro.

## Categoria

Cadeia de caracteres

## Sintaxe da função

`upper(<parameters>)`

## Assinatura e tipo retornado

`upper(<string>)`

Retorna uma string.

## Exemplo

`upper("b")`

Retorna &quot;B&quot;.
