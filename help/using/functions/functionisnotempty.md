# isNotEmpty {#isNotEmpty}

Retorna true se a string no parâmetro não estiver vazia.

## Categoria

String

## Sintaxe da função

`isNotEmpty(<parameters>)`

## Parâmetros

* string

## Assinatura e tipo retornado

`isNotEmpty(<string>)`

Retorna um booleano.

## Exemplo

`isNotEmpty("")`

Retorna false.

`isNotEmpty("hello")`

Retorna true.
