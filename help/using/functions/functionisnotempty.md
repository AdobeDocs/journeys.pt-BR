# isNotEmpty {#isNotEmpty}

Retorna verdadeiro se a string no parâmetro não estiver vazia.

## Categoria

String

## Sintaxe da função

`isNotEmpty(<parameters>)`

## Parâmetros

* sequência de caracteres

## Assinatura e tipo retornado

`isNotEmpty(<string>)`

Retorna um valor booleano.

## Exemplo

`isNotEmpty("")`

Retorna falso.

`isNotEmpty("hello")`

Retorna verdadeiro.
