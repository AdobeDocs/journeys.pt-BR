# length {#length}

Retorna o número de caracteres da expressão de string no parâmetro.

## Categoria

Cadeia de caracteres

## Sintaxe da função

`length(<parameters>)`

## Parâmetro

* string

## Assinatura e tipo retornado

`length(<string>)`

Retorna um número inteiro.

## Exemplo

`length("Hello World")`

Retorna 11.
