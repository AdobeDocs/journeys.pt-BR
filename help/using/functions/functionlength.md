# length {#length}

Retorna o número de caracteres da expressão da string no parâmetro .

## Categoria

String

## Sintaxe da função

`length(<parameters>)`

## Parâmetro

* string

## Assinatura e tipo retornado

`length(<string>)`

Retorna um número inteiro.

## Exemplo

`length("Hello World")`

Retorna 11.
