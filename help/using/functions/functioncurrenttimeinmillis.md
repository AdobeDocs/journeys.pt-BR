# currentTimeInMillis {#currentTimeInMillis}

`currentTimeInMillis()`

## Parâmetros

Essa função não usa parâmetros.

## Assinaturas e tipo retornado

`currentTimeInMillis()`

Retorna um número inteiro.

## Exemplos
