# Terminar atividade{#section_vqp_4ft_dgb}

A atividade **[!UICONTROL End]** permite marcar o final de cada caminho da jornada. Não é obrigatório, mas recomendado para clareza visual. Na verdade, se a jornada tiver várias atividades end, recomendamos adicionar um rótulo a cada end para facilitar a leitura dos relatórios. Consulte [esta página](../reporting/about-journey-reports.md).
